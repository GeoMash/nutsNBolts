Help

I only covered the most important components of this theme. If you have any questions, feel free to contact me through the contact form that {wrap}bootstrap provides

Be sure to mention the License ID you got through your purchase

Have you found a bug in this theme? Send me a message and let me know what your issue is. I'll do my best to help you

What’s covered by support:

  1. Upgrades to latest version of Twitter Bootstrap
  2. Bug fixes for components created by me
  3. Issues related to cross browser support in my themes
  4. Limited how-to advice and customization direction

What’s not covered by support:

  • Fixing errors or bugs with Twitter Bootstrap
  • In-depth theme customization
  • Addition of functionality not included by default
  • Any errors relating to theme customization.
